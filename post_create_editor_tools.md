# Post Creation Editor Tools

This document lists all text editing tools and formatting features implemented in the post creation editor, with a brief description of each tool.

## Text Formatting Tools

- **Bold**: Makes selected text bold.
- **Italic**: Italicizes selected text.
- **Underline**: Underlines selected text.
- **Strikethrough**: Strikes a line through selected text.
- **Subscript**: Formats text as subscript.
- **Superscript**: Formats text as superscript.

## Font Controls

- **Font Family**: Change the font family (Inter, Fraunces, Georgia, Arial, Times New Roman).
- **Font Size**: Change the font size (12px, 14px, 16px, 18px, 20px, 24px, 28px, 32px, 36px, 48px).
- **Text Color**: Change the color of the text.
- **Background Color**: Change the background color of the text.

## Headings

- **H1**: Heading 1 style.
- **H2**: Heading 2 style.
- **H3**: Heading 3 style.

## Alignment

- **Align Left**: Align text to the left.
- **Align Center**: Center align text.
- **Align Right**: Align text to the right.
- **Justify**: Justify text.

## Lists

- **Bullet List**: Unordered (bulleted) list.
- **Numbered List**: Ordered (numbered) list.

## Block Elements

- **Blockquote**: Formats text as a blockquote.
- **Code Block**: Formats text as a code block (preformatted text).

## Insert Tools

- **Link**: Insert a hyperlink.
- **Image**: Insert an image.
- **Table**: Insert a table.
- **Horizontal Rule**: Insert a horizontal line.

## Undo/Redo

- **Undo**: Undo the last action.
- **Redo**: Redo the last undone action.

## Utilities

- **Scripture Lookup**: Opens a scripture lookup tool.

---

*This list is based on the implementation in the post creation editor as of February 27, 2026.*
